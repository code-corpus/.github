## 📖 About this organization

This organization hosts the **Production Code Corpus** project: a collection of carefully built, professional software repositories that together form a high-quality code dataset. Every repository here is **authored from scratch** to a strict engineering standard real, working code with documentation, tests, automation, and a clean development history.

We don't collect code from the wild. **We build it properly.**

## ✅ Engineering standards

Every repository in this organization is built to the same bar:

- 📄 Clear **README** and project documentation
- 🧪 Real **automated tests**
- 🔄 **CI/CD pipeline** on every repo (lint · test · build)
- 🔐 **Secret scanning** (TruffleHog) in CI and pre-commit
- 💬 Well-commented, readable code
- 🌿 Clean Git history every change via **Issue → branch → Pull Request → review → merge**

---

## 🛠️ How we work

```
Issue  →  feature branch  →  Pull Request  →  review + green CI  →  merge  →  delete branch
```

Each repository is owned end to end by one engineer, developed through small, reviewed pull requests. Direct pushes to `main` are not allowed quality is enforced automatically.
